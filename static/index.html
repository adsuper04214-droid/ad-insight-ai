Gemini API Key 설정

이 서비스는 광고 상세 리포트를 기획 관점에서 실시간 분석하기 위해 Google Gemini API를 사용합니다. 입력하신 API 키는 브라우저 로컬 저장소(localStorage)에만 안전하게 저장되며 외부로 전송되지 않습니다.

Gemini API Key 발급 방법

  1. Google AI Studio에 접속하여 로그인합니다.
  2. Get API key 버튼을 클릭합니다.
  3. 새 프로젝트에서 API Key를 발급받아 복사한 뒤 위 입력창에 붙여넣기 하세요. (무료 한도로 충분히 테스트 가능합니다)
